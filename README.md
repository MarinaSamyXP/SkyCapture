# 🚁 DJI Mavic 4 Pro Landing Page

A modern, responsive product landing page showcasing the **DJI Mavic 4 Pro** drone. This project highlights product features, specifications, customer reviews, and related accessories in a visually engaging layout.

---

## 📌 Overview

This project is a **front-end web design** built using HTML and CSS. It simulates a real e-commerce product page for a high-end drone, focusing on clean UI, structured layout, and user-friendly navigation.

---

## ✨ Features

- 🧭 **Responsive Navigation Bar**
  - Smooth scrolling to sections (Home, Specifications, Reviews, Products)

- 🛍️ **Hero Section**
  - Product image with discount badge
  - Pricing details and call-to-action buttons

- 📊 **Comparison Table**
  - Compare DJI Mavic 4 Pro with competitors
  - Highlights key specs like:
    - Video resolution (8K)
    - Flight time (40 mins)
    - AI obstacle avoidance

- ⭐ **Ratings & Reviews**
  - Overall rating breakdown
  - Customer testimonials
  - Verified purchase indicators

- 🎒 **Related Products**
  - Accessories like controller, battery kit, travel case, safety kit

- 📱 **Footer Section**
  - Company info
  - Product links
  - Support links
  - Social media icons

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the page  
- **CSS3** – Styling and layout  
- **Font Awesome** – Icons  
- **Google Fonts (Exo)** – Typography
